## 核心改动

### 🚫 全局禁用 TIER_B（普通叙事）

**数据支撑**:
- TIER_B: 52笔交易，19.2%胜率，-$83.99总亏损
- SOL + TIER_B: 29笔交易，10.3%胜率，-37.51%平均亏损（灾难级）
- BSC + TIER_B: 23笔交易，30.4%胜率，-6.54%平均亏损（可接受但仍亏损）

**修改文件**:

1. `src/config/strategy.js`
   - HARD_GATES 新增 `FORBIDDEN_TIERS: ['TIER_B']`
   - DEFENSIVE_MODE.ALLOWED_TIERS 从 `['TIER_A', 'TIER_B']` 改为 `['TIER_A']`

2. `src/risk/risk-manager.js`
   - 新增全局TIER检查逻辑（独立于防御模式，始终生效）
   - 位置：evaluateSignal() 函数，基础分数检查之后

## 回滚方法

如需回滚到v9.4：
1. `HARD_GATES.FORBIDDEN_TIERS` 删除或设为 `[]`
2. `DEFENSIVE_MODE.ALLOWED_TIERS` 改回 `['TIER_A', 'TIER_B']`
3. 删除 risk-manager.js 中的全局TIER检查逻辑（第311-322行）
